There are two established classes of methods for object detection in images
* sliding windows
  * Deformable Part Model (DPM)
  * OverFeat
  * YOLO
  * SSD
  * G-CNN

* region proposal classification
  * MultiBox
  * R-CNN
  * SPPnet
  * Fast R-CNN
  * ION
  * Faster R-CNN

* bounding box
  * Selective Search
  * LocNet
